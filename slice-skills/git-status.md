# Git Status Skill

When invoked, perform these steps:

1. Run `git status` to show the current repository state
2. Run `git log --oneline -5` to show the last 5 commits
3. If there are uncommitted changes, list them clearly
4. Summarize the current state of the repository in a concise way

Be clear and organized in your output.
